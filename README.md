Deskripsi Proyek:

Kampoenga Kadjoetangan Heritage merupakan sebuah destinasi wisata tempo dulu yang terletak di Kota Malang. Wisata ini cukup dikenal dan bahkan telah menjadi salah satu ikon di kota tersebut.

Kampoeng Kadjoetangan Heritage: Integrated Tourism Platform merupakan sebuah platform web yang berfokus pada penyampaian informasi dan kemudahan akses bagi wisatawan yang hendak atau saat ini berkunjung ke Kampoeng Kadjoetangan Heritage. Situs web ini berfokus pada fungsi-fungsi fundamental web artikel dengan sedikit penambahan fungsionalitas dari segi e-commerce. Adapun fundamental artikel dari web ini seperti menyediakan halaman informasi seputar "apa yang ada" di Kampoeng Kadjoetangan Heritage. Sedangkan secara bisnis--cukup bersentuhan dengan proses bisnis--, web ini menyediakan menu pemesanan dan pembayaran tiket secara realtime dengan bantuan sistem.
